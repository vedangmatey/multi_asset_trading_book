# How to Use the Dashboard

## Market Data Page
- Select an asset basket
- Choose a date range
- View prices and factor series

## Correlations Page
- Explore cross-asset correlations
- Identify diversification benefits

## VaR Page
- Historical VaR computation
- Confidence-level selection
- Portfolio-level risk interpretation

## Interpretation
- Equity/FX: returns-based factors
- Rates: yield changes in basis points
